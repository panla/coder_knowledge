# 安装 Postman

## 下载

[下载地址](https://www.postman.com/downloads/)

## desktop

```text
[Desktop Entry]
Type=Application
Categories=Development
Encoding=UTF-8
Name=Postman
Exec=/home/opt/Postman/app/Postman
Icon=/home/opt/Postman/app/resources/app/assets/icon.png
Terminal=false
```
