# 数据卷

## 命令

```bash
docker volume --help

# 查看 查看数据卷
docker volume ls

# 详细信息
docker volume inspect volume
```

## 特点

- 数据卷可以在容器之间共享和重用
- 对数据卷的修改会立即影响对应容器
- 对数据卷的更新不会影响原始镜像

## 宿主机与容器的数据卷

```bash
docker run -it -v /srv/data:/container:/data centos
-v /root/apps:/usr/local/apps:ro 只读，容器只读，宿主机可以影响容器，反之不可以
```
