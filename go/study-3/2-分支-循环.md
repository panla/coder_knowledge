## 分支

### if else

```go
func main() {
    n := 10

    if n == 5 {
        fmt.Println(n)
    } else if n == 6 {
        fmt.Println(n)
    } else {
        fmt.Println(n)
    }
}
```

### switch

```go
func main() {
    n := 10

    switch n {
        case n == 1:
            fmt.Println(n)
        case n == 2:
            fmt.Println(n)
        default:
            fmt.Println(n)
    }
}
```

```go
func main() {
    v := 10

    switch x := v.(type){
        case uint8:
            fmt.Println(x)
        case uint16:
            fmt.Println(x)
    }
}
```

## 循环

### for 循环

```go
func main() {
    n := 10

    for n > 0 {
        n--
        fmt.Println(n)
    }
}
```

```go
func main() {
    for n := 10; n > 0; n-- {
        fmt.Println(n)
    }
}
```

```go
func main() {

    n := 10
    for ; n > 0; n-- {
        fmt.Println(n)
    }
}
```

### 对字符串循坏

go语言中的字符串实际上是类型为byte的只读切片。或者说一个字符串就是一堆字节。
这意味着，当我们将字符存储在字符串中时，实际存储的是这个字符的字节

```go
func main() {
    // Unicode 遍历
    str := "超星系团"

    for pos, char := range str {
        fmt.Println(pos, char)

        // 打印 uint8--byte
        // 0 36229
        // 3 26143
        // 6 31995
        // 9 22242
    }

    // UTF-8 遍历
    for i := 0; i < len(str); i++ {
        fmt.Println(i, str[i])

        // 打印 int32---rune
        // 0 232
        // 1 182
        // 2 133
        // 3 230
```

### 对 map 循环

```go
func main() {
     info := map[string]int{
        "aaa": 100,
        "bbb": 120,
    }


    for key, value := range info{
        fmt.Println(key, value)

        // 打印
        // aaa 100
        // bbb 120
    }
}
```
