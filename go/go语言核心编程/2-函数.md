# 函数

```text
函数，一种类型，可以作为参数传递，或返回值
多值返回
闭包
支持可变参数
```

## 1 基本概念

### 1.1 函数定义

函数的定义包括：关键字，函数名，参数列表，返回列表，函数体

首字母的大小写决定该函数在其他包的可见性：大写时其他包可见，小写时只有相同的包可以访问

- 不支持 默认值参数
- 不支持 函数重载

```go
func func_add(a, b int8) int8 {
    return a + b
}
```

### 1.2 多值返回

```go
func swap(a, b int) (int, int){
    return b, a
}
```
