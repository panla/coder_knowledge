# 映射

[toc]

## 0

存储无序键值对

使用散列表

## 1 声明与初始化

### 1.1 make

```go
dict := make(map[string]int)
// map[]

// 可以赋值
```

### 1.2 使用字面量

```go
dict := map[string]string{"Red": "a", "Green": "b"}
// map[Green:b Red:a]
```

### 1.3 空映射

```go
dict := map[string]int{}
// map[]

// 可以赋值
```

### 1.4 nil

```go
var dict map[string]int
// == nil
// map[]，此时未初始化，?????用在哪里

// 不可赋值
```

## 2 use map

### 2.1 赋值

```go
dict := map[string]int{}

dict["Red"] = 1
```

### 2.2 get

```go
dict := map[string]int{"Red": 1, "Green": 2}

a, exists := dict["Red"]
// 1 true

b := dict["Red"]
// 1
```

### 2.3 range

```go
dict := map[string]int{"Red": 1, "Green": 2}

for key, value := range dict {
    fmt.Println(key, value)
}
// Red 1
// Green 2
```

### 2.4 delete

```go
dict := map[string]int{"Red": 1, "Green": 2}

delete(dict, "Red")
```
